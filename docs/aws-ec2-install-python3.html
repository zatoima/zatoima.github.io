EC2(Amazon Linux)にpip、python3をインストール

2019/10/18 2 min read Python, EC2

0.コマンド

pip は Python 3.4 以降では Python 本体に同梱されるようなっているのでpip3の個別のインストールは不要。

sudo yum install -y python3
