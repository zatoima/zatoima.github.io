S3 Event Missed Notifications Do Not Occur as of 2022

Published on 2022/07/01 üìñ 1 min read
AWS S3

Conclusion

As stated in the title, S3 event missed notifications no longer occur.

Earlier documentation noted that S3 event notifications could be lost, and this was even mentioned in the documentation. However, as of 2022, this is no longer the case as described below.

https://docs.aws.amazon.com/en_us/AmazonS3/latest/userguide/NotificationHowTo.html

Amazon S3 event notifications are designed to be delivered at least once. Typically, event notifications are delivered in seconds but can sometimes take a minute or longer.

Some older articles mentioned that S3 event fires could be missed. Since there were not many articles written about this specification improvement, here is a note about it.

https://dev.classmethod.jp/articles/s3-event-notification-vs-eventbridge-event-processing-overflow-rate/
