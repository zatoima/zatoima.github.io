GCP Associate Cloud Engineer Exam Passed

Published on 2021/09/14 üìñ 1 min read
GCP

I’m currently on parental leave, dealing with household chores and childcare, which makes it difficult to find concentrated study time. However, I felt that if I didn’t study something, it would be psychologically hard to return to work after parental leave, so I found pockets of time to study GCP. My motivation was preparation for obtaining the GCP Professional Data Engineer certification.

Exam Overview

Associate Cloud Engineer Certification | Google Cloud https://cloud.google.com/certification/cloud-engineer

Exam Guide

Associate Cloud Engineer Certification Exam Guide | Google Cloud https://cloud.google.com/certification/guides/cloud-engineer

Candidate’s Cloud/GCP Knowledge

  • AWS 12 certifications
  • AZ-900: Microsoft Azure Fundamentals
  • GCP experience: Was attracted by GCE’s Always Free tier and deployed WordPress, then left it
  • Studied BigQuery and Cloud Spanner by reading seminar materials

Study Time

Approximately 2 weeks

Study Steps

I went through Coursera and Qwiklabs quickly.

  1. Read the book “GCP Textbook”
  2. Read the book “Google Cloud Platform Enterprise Design Guide”
  3. Try the Associate Cloud Engineer Practice Exam
  4. Qwiklabs Quests
    1. Google Cloud Essentials
    2. Cloud Engineering
    3. Baseline: Infrastructure
    4. Cloud Architecture
  5. Coursera
    1. Architecting with Google Compute Engine Japanese Specialization
      1. Google Cloud Platform Fundamentals: Core Infrastructure Japanese
      2. Essential Cloud Infrastructure: Foundation Japanese
      3. Essential Cloud Infrastructure: Core Services Japanese
      4. Elastic Cloud Infrastructure: Scaling and Automation Japanese
    2. Preparing for the Google Cloud Associate Cloud Engineer Exam
  6. Try Udemy practice questions
  7. Check related official documentation from GCP Associate Cloud Engineer Cheat Sheet
  8. [Master] GCP Getting Started JP
    1. GCPUG public materials
  9. GCP Tutorials
  10. Read main service session materials from Cloud OnAir

Thoughts

  • I hadn’t studied much about moving resources and data between projects, and the permissions involved, so I was a bit panicked. I felt the exam trends were different from the practice exam.
  • It wasn’t just a knowledge-based exam; I felt it was asking about things that would actually come up in real work.
  • It was interesting to study the differences in design philosophy and service lineup compared to AWS.
  • Next up is GCP Professional Data Engineer.
