Exporting AWS Resource Information Using awsets

Published on 2020/10/05 üìñ 1 min read
AWS EC2 Awsets

As the description says, A utility for crawling an AWS account and exporting all its resources for further analysis. ‚Äî this tool exports all AWS resources associated with an AWS account in JSON format.

GitHub - trek10inc/awsets: A utility for crawling an AWS account and exporting all its resources for further analysis. https://github.com/trek10inc/awsets

Here are my notes from trying out the latest version at the time, 0.4.1. If you already have an environment where AWS CLI can run, installation and execution is quick and easy.

Installation

cd /home/ec2-user
mkdir awsets
cd awsets
wget https://github.com/trek10inc/awsets/releases/download/v0.4.1/awsets_0.4.1_linux_x86_64.tar.gz
tar xvfz awsets_0.4.1_linux_x86_64.tar.gz

export PATH=PATH=$PATH:/home/ec2-user/awsets/

Execution

awsets list --regions ap-northeast-1 -o ec2.json --include ec2
awsets list --regions ap-northeast-1 -o elasticsearch.json --include elasticsearch
awsets list --regions ap-northeast-1 -o ecs.json --include ecs

There are many more features available. See the references and the official GitHub page.
