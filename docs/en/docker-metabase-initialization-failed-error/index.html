Set environment variables to use the locally built PostgreSQL instead of the H2 database.

Migrating to a production application database 

export MB_DB_TYPE=postgres
export MB_DB_CONNECTION_URI="jdbc:postgresql://localhost:5432/metabase?user=postgres&password=postgres"
java -jar metabase.jar

Started successfully.
