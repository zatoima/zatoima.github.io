Installing Oracle Database 19c on Oracle Linux in VirtualBox

2019/06/22Published on 📖 1 min read
🔴Oracle

Introduction

In the previous article, I completed the preparation of required packages, users, and directories, so now I will actually install Oracle Database 19c and build the DB. This is a single-instance environment. The OS is Oracle Linux 7.6.

Pre-installation Guide

Pre-Installation Preparation for Oracle Database 19c | my opinion is my own https://zatoima.github.io/oracle-pre-install-19c.html

Oracle Database 19c for Linux Installation Guide

Oracle Database Database Installation Guide, 19c for Linux https://docs.oracle.com/cd/F19136_01/ladbi/index.html

Installing Oracle Database 19c

First, download “Oracle Database 19.3.0.0.0 (V982063-01.zip)” from “Oracle Software Delivery Cloud ”.

Starting from Oracle 18c, the installation method has changed from using runinstaller to install to ORACLE_HOME, to an unzip format.

[oracle@dbsrv dbhome_1]$ ls -l V982063-01.zip

[oracle@dbsrv dbhome_1]$

Creating the DB

Create the DB using DBCA as before.

During DBCA execution, run orainstRoot.sh and root.sh as with previous versions.

Setting Environment Variables

export TMPDIR=$HOME/tmp
export TEMP=$HOME/tmp
export ORACLE_BASE=/u01/app/oracle
export ORACLE_HOME=/u01/app/oracle/product/19.0.0/dbhome_1
export PATH=$ORACLE_HOME/bin:$ORACLE_HOME/jdk/bin:${PATH}
export LD_LIBRARY_PATH=$ORACLE_HOME/lib
export NLS_LANG=JAPANESE_JAPAN.UTF8
export LANG=ja_JP.UTF-8
