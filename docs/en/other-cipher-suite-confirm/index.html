How to Investigate Cipher Suites Used for Server Communication

Published on 2022/04/14 📖 1 min read
AWS Other

A quick note for reference. In this example, I’ll look into various details about S3’s s3.amazonaws.com.

What Is a Cipher Suite?

This site provides a clear explanation:

https://pkiwithadcs.com/cipher_suite/

Checking Available Cipher Suites

nmap --script ssl-enum-ciphers -p 443 s3.amazonaws.com

Checking with SSL Server Test

You can see the priority order of which cipher suites are used.

https://www.ssllabs.com/ssltest/analyze.html?d=s3.amazonaws.com

Mapping Between Cipher Suite Names and OpenSSL Names

OpenSSL cipher suite names do not match the commonly used names, so you need to refer to a mapping table to identify which is which.

https://www.ipa.go.jp/security/ipg/documents/tls_cipher_suite_config_20200707.pdf
