Installing pip and python3 on EC2 (Amazon Linux)

Published on 2019/10/18 📖 1 min read
Python EC2

0. Command

Since pip has been bundled with Python itself since Python 3.4, a separate installation of pip3 is not required.

sudo yum install -y python3
