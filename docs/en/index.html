I purchased the [ThinkPad X13 Gen2 AMD](https://www.lenovo.com/jp/ja/notebooks/thinkpad/x-series/X13-AMD-G2/p/22TPX13X3A1). It cost 168,828 yen.

## Reasons for the Purchase

- The lightweight body at approximately 1.19kg makes it easy to carry around
- I wanted to use the Ryzen 7 PRO 5850U with Zen 3 architecture — 8 cores and 16 threads!
- Even with the memory and SSD customizations listed below, the overall price felt reasonable
  - 32GB memory — I've heard LPDDR4X is excellent
  - 512GB SSD storage
- It comes with a Type 2280 M.2 SSD, but when you select the 4G/5G option, there's a slot for a wireless communication module that accepts a Type 2242 M.2 SSD. I have a Type 2242 M.2 SSD from a previous laptop that I'd like to put to good use (this modification is done at your own risk)
  - I had no issues doing this with my T470s
- I love Lenovo ThinkPads. I love the TrackPoint

## Drawbacks

- It might just be something I need to get used to, but I can't get comfortable with the position of the Ctrl and Fn keys on ThinkPads. I feel like I'd need to switch all my computers and keyboards to Lenovo to adapt.
  - I could remap the keys, but that feels wrong

## Specifications

- Processor: AMD Ryzen 7 PRO 5850U (1.90GHz, 16MB L3)
- Pre-installed OS: Windows 10 Home 64bit
- OS Language: Windows 10 Home 64bit - Japanese
- Memory: 32 GB LPDDR4X-4266MHz (Onboard)
- 1st SSD: 512GB Solid State Drive (M.2 2280, PCIe-NVMe) OPAL compatible
- Display: 13.3" WUXGA (1920x1200) IPS, Anti-glare, 300nit, Narrow bezel, Non-touch, 72% NTSC
- Body Color: Black
- Top Cover: PPS (Polyphenylene Sulfide)
- Graphics: Integrated Graphics
- Built-in Camera: 720p HD Camera (with microphone)
- Wireless LAN Adapter: Intel Wi-Fi 6 AX200 (2x2) + Bluetooth (vPro compatible)
- WWAN: Quectel EM120R-GL 4G LTE CAT12
- Fingerprint Sensor: Fingerprint Sensor (Black)
- Pointing Device: WWAN, No NFC, Black, Magnesium
- Keyboard: Japanese Keyboard
- Card Slot: No Smart Card Reader
- TPM Setting: TPM enabled (TCG V2.0 compliant, hardware chip)
- Absolute BIOS Selection: BIOS Absolute enabled
- Battery: 4-cell Lithium-ion Battery (54.7Wh)
- Power Adapter: 65W AC Adapter (2-pin) (USB Type-C)
- Display Panel: 13.3" WUXGA (1920x1200) IPS, Anti-glare, 300nit, Narrow bezel, Non-touch, 72% NTSC, 720p HD Camera, Microphone, No WWAN support, FreeSync, PPS, Black
- Language Pack: Japanese
- Standard Warranty: 1-year Depot Repair
