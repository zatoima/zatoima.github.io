I’ve set up GitHub contributions to display on the top page. Since I run this blog on GitHub Pages, the “grass” grows whenever I write a blog post.

I did this out of motivation and simple aesthetic appeal. Since this blog runs on GitHub Pages, the grass grows whenever I write a post.

2.) Enter your GitHub ID

3.) A meta tag is generated; take note of it

4.) Add a tag so that clicking the image takes you to your GitHub page
<a href="https://github.com/zatoima" target="_blank">
  <img src="https://grass-graph.appspot.com/images/zatoima.png">
</a>
5.) Paste this tag in the designated location in your blog service

I pasted it on the top page.
