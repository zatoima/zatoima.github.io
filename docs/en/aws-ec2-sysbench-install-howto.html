Installing the RDBMS Benchmark Tool sysbench on EC2 (Amazon Linux)

2019/10/21Published on 📖 1 min read
☁️AWS 💻EC2

Commands to Run

sysbench exists in the EPEL repository, so it needs to be installed in advance.

sudo -s
wget https://dl.fedoraproject.org/pub/epel/7/x86_64/Packages/e/epel-release-7-12.noarch.rpm
rpm -ivh epel-release-7-12.noarch.rpm
yum -y install epel-release

yum -y install sysbench
