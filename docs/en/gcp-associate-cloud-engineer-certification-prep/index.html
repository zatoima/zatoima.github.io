Study Steps

  1. Read the book “GCP Textbook”
  2. Read the book “Google Cloud Platform Enterprise Design Guide”
  3. Try the Associate Cloud Engineer Practice Exam
  4. Qwiklabs Quests
    1. Google Cloud Essentials
    2. Cloud Engineering
    3. Baseline: Infrastructure
    4. Cloud Architecture
  5. Coursera
    1. Architecting with Google Compute Engine Japanese Specialization
      1. Google Cloud Platform Fundamentals: Core Infrastructure Japanese
      2. Essential Cloud Infrastructure: Foundation Japanese
      3. Essential Cloud Infrastructure: Core Services Japanese
      4. Elastic Cloud Infrastructure: Scaling and Automation Japanese
    2. Preparing for the Google Cloud Associate Cloud Engineer Exam
  6. Try Udemy practice questions
  7. Check related official documentation from GCP Associate Cloud Engineer Cheat Sheet
  8. [Master] GCP Getting Started JP
    1. GCPUG public materials
  9. GCP Tutorials
  10. Read main service session materials from Cloud OnAir

Additional Steps

  • Study Kubernetes. Look it up online to get a general understanding
  • Migrate a self-made app running on EC2 to GCP targeting Always Free
    • The EC2 instance cost of a t2.nano running even after the free period ended was adding up
  • Try out BigQuery, Cloud SQL, and Spanner
