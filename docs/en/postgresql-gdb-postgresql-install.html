Building PostgreSQL 10 with Debug Support from Source Code and Using gdb

2020/04/03Published on 📖 1 min read
🐘PostgreSQL

This article assumes PostgreSQL 10.7.

Pre-checks

GNU make Version

Confirm it is version 3.80 or higher.

make --version

Install gcc

sudo yum -y install gcc

readline-devel Package

sudo yum -y install readline-devel

zlib-devel Package

sudo yum -y install zlib-devel

Create OS User

groupadd -g 1101 postgres
useradd -u 1101 -g postgres -G postgres -d /var/lib/pgsql postgres
passwd postgres

Change Permissions

chown -R postgres:postgres /var/lib/pgsql
chmod -R 755 /var/lib/pgsql

Add the following to environment variables:

export PGDATA=/var/lib/pgsql/10/data

Install PostgreSQL

Download & Extract Source

su - postgres
wget https://ftp.postgresql.org/pub/source/v10.7/postgresql-10.7.tar.gz
tar xvfz postgresql-10.7.tar.gz

Build

cd $HOME/postgresql-10.7
./configure  --enable-debug --prefix=/var/lib/pgsql/10
make

All of PostgreSQL successfully made. Ready to install. will be output.

Install

make install

PostgreSQL installation complete. will be output.

Initialize PostgreSQL

Initialize the DB

mkdir -p /var/lib/pgsql/10/data
export PGDATA=/var/lib/pgsql/10/data
whoami
$HOME/10/bin/initdb --pgdata=$PGDATA
$HOME/10/bin/pg_ctl start
$HOME/10/bin/pg_ctl status

Connect and Verify Version

Set up environment variables and PATH, then connect.

[postgres@postpub ~]$ psql
psql (10.7)
Type "help" for help.

postgres=# select version();

postgres=# select pg_backend_pid();
