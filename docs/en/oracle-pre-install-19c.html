Pre-Installation Preparation for Oracle Database 19c

Published on 2019/06/14 📖 2 min read
Oracle

Introduction

In the previous article, I installed Oracle Linux 7.6 as a prerequisite for installation, so this time I will continue from there. This covers pre-installation preparation for DB installation and DB setup.

The installation guide for Oracle Database 19c for Linux is here:

Oracle Database Database Installation Guide, 19c for Linux https://docs.oracle.com/cd/F19136_01/ladbi/index.html

Changing the Hostname

# Change hostname
hostnamectl set-hostname dbsrv.oracle.jp
  • Exit the session and log in again
[root@localhost ~]# hostnamectl set-hostname dbsrv.oracle.jp
[root@localhost ~]#
[root@localhost ~]# echo $?
0
[root@localhost ~]#

systemctl disable firewalld

Adding Users and Groups

Since this is a single-instance environment rather than RAC, users for the RAC environment are commented out.

groupadd -g 54321 oinstall
groupadd -g 54322 dba
groupadd -g 54323 oper
#groupadd -g 54324 backupdba
#groupadd -g 54325 dgdba
#groupadd -g 54326 kmdba
#groupadd -g 54327 asmdba
#groupadd -g 54328 asmoper
#groupadd -g 54329 asmadmin
#groupadd -g 54330 racdba

useradd -u 54321 -g oinstall -G dba,oper oracle
