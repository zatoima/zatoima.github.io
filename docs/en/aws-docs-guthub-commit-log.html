Checking AWS Documentation Update History on GitHub and Getting RSS Feeds

Published on 2020/09/04 üìñ 1 min read
AWS Aurora

I just recently learned that AWS documentation can be viewed on GitHub. It appears it became open source in 2018.

AWS Documentation is now open source and available on GitHub | Amazon Web Services Blog https://aws.amazon.com/jp/blogs/news/aws-documentation-is-now-open-source-and-on-github/

You can access user guides and developer guide repositories from the following URL:

https://github.com/awsdocs

For example, the repository for the Aurora User Guide is here:

https://github.com/awsdocs/amazon-aurora-user-guide

Since it’s on GitHub, you can view commit logs and see exactly what parts changed.

https://github.com/awsdocs/amazon-aurora-user-guide/commit/f83c1cce2f8fa9bbb7bc4c72ac2233c381f9bd84

GitHub can output various atom feeds, so you can subscribe to RSS and check whenever there are updates. For example, the feed for the Aurora User Guide commit log is as follows. By changing the repository name to a different one, you can receive updates for any repository you’re interested in.

https://github.com/awsdocs/amazon-aurora-user-guide/commits/master.atom
