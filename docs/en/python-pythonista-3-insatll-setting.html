I had some Python code I didn’t need to run regularly, but wanted to run occasionally during free time. It would be fine if I always had access to a PC when I wanted to run it, but that’s not always the case, so I tried Pythonista 3 to see if I could run it from my iPhone or iPad that I always carry.

Pythonista 3 is a paid app for running Python on iOS. As of January 2020, it costs 1,200 yen.

Popular third-party modules like numpy, matplotlib, and requests, as well as iOS-specific customized modules are available. You can access iPhone motion sensor data, photo library, contacts, alarms, iOS clipboard, and other information.

My concern is whether there will be continued updates. (The last update was in 2018.)

That said, I couldn’t find any other option that lets you install third-party modules on iOS and run Python code, so this seems to be the only choice.

This time, I’ll install StaSh to create a pip-installable environment and use tweepy to operate the Twitter API.

Installing StaSh

StaSh (Pythoni**sta Sh**ell) is an extension that enables pip and other command operations in Pythonista 3. Since pip is an essential feature for Python use, you should install StaSh as well.

To install, first open the console and run the following command. This command downloads getstash.py from https://raw.githubusercontent.com/ywangd/stash/master/getstash.py.

import requests as r; exec(r.get('http://bit.ly/get-stash').text)

launch_stash.py is placed there; run this Python file.

The console screen launches. Run pip commands from this screen. Here, we install tweepy, a wrapper for the Twitter API.
