SAMテンプレートを修正

下記を追加

  AutoPublishAlias: live
  DeploymentPreference:
    Type: Linear10PercentEvery10Minutes

[ec2-user@bastin sam-app]$ cat template.yaml 
AWSTemplateFormatVersion: '2010-09-09'

  Sample SAM Template for sam-app

  HelloWorldFunction:
