IIJmio offers a call forwarding service. I had a situation where I needed to use it, so here are my notes.

# Reference

> https://www.iijmio.jp/hdd/miofone/voiceopt.html#transfer

# Notes

- The call charges from the contracted mobile phone to the forwarding destination are billed to the call forwarding service subscriber.
- Charges from the caller are also required.
- When call forwarding is set up, voicemail becomes disabled.
- You can specify the number of seconds before forwarding (default is 7 seconds).
- After setting up forwarding, an announcement "This call will be forwarded, please wait a moment" plays.
  - Change the guidance settings as needed.

# Steps

To change the forwarding time to 5 seconds and disable the guidance announcement, the following steps are required:

> https://www.iijmio.jp/hdd/miofone/voiceopt.html#transfer

1. Register/change the forwarding destination phone number
2. Start the call forwarding service
3. Set guidance to none
4. Set the ring time to "5 seconds"
